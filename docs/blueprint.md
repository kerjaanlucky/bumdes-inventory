# **App Name**: InventoryFlow

## Core Features:

- Master Barang Management: Admin can create, read, update, and delete (CRUD) master data for items, each associated with a specific branch.
- Stock Management: Track stock levels for each item at each branch. Display current stock, stock history, and trigger alerts for low stock levels.
- Purchase Management: Record purchase orders and track incoming stock. Link purchases to specific branches and update stock levels automatically.
- Sales Management: Record sales transactions, deduct sold items from stock at the corresponding branch, and generate invoices.
- Reporting: Generate reports on stock levels, purchase history, and sales data, filtered by branch and date range.
- Branch Management (Admin): Admin role can add, remove, and manage branches.
- User Management (Admin): Admin can add users, assign roles (admin or regular user), and link users to specific branches.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey trust and reliability.
- Background color: Very light gray (#F5F5F5), nearly white, for a clean and professional look.
- Accent color: Soft teal (#4CAF50) to indicate growth and positive action.
- Body and headline font: 'PT Sans', a humanist sans-serif, provides a balance of modernity and approachability for readability.
- Use clear and consistent icons from a standard library (e.g., Material Design Icons) to represent different item categories, stock actions, and report types.
- Implement a responsive layout to ensure usability across different devices, with a clear hierarchy of information.
- Subtle transitions and loading animations to enhance the user experience without being distracting.
